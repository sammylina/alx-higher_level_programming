The my-list module
==================

using print_sorted
------------------

This is python doctest module to test a function that prints a
sorted list

    >>> MyList = __import__('1-my_list').MyList

    sort same values of list
    >>> ml = MyList()
    >>> ml.append(3)
    >>> ml.append(3)
    >>> ml.append(3)
    >>> ml.print_sorted()
    [3, 3, 3]

    sort different values of list
    >>> ml.append(4)
    >>> ml.append(1)
    >>> ml.print_sorted()
    [1, 3, 3, 3, 4]

    sort reversely sorted list
    >>> ml = MyList()
    >>> ml.append(4)
    >>> ml.append(3)
    >>> ml.append(2)
    >>> ml.print_sorted()
    [2, 3, 4]
