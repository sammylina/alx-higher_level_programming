The my-list module
==================

documentation check
-------------------


using print_sorted
------------------

This is python doctest module to test a function that prints a
sorted list

    >>> MyList = __import__('1-my_list').MyList


    same values of list

    >>> ml = MyList()
    >>> ml.append(3)
    >>> ml.append(3)
    >>> ml.append(3)
    >>> ml.print_sorted()
    [3, 3, 3]


    different values of list

    >>> ml = MyList()
    >>> ml.append(3)
    >>> ml.append(4)
    >>> ml.append(1)
    >>> ml.append(-3)
    >>> ml.print_sorted()
    [-3, 1, 3, 4]

    reversely sorted list

    >>> ml = MyList()
    >>> ml.append(4)
    >>> ml.append(3)
    >>> ml.append(2)
    >>> ml.print_sorted()
    [2, 3, 4]

    empty list
    
    >>> ml = MyList()
    >>> ml.print_sorted()
    []

    negative numbers

    >>> ml = MyList()
    >>> ml.append(-4)
    >>> ml.append(-1)
    >>> ml.append(-5)
    >>> ml.print_sorted()
    [-5, -4, -1]

    single value only

    >>> ml = MyList()
    >>> ml.append(4)
    >>> ml.print_sorted()
    [4]

    positive and negative values

    >>> ml = MyList()
    >>> ml.append(1)
    >>> ml.append(0)
    >>> ml.append(-1)
    >>> ml.append(11)
    >>> ml.append(-31)
    >>> ml.print_sorted()
    [-31, -1, 0, 1, 11]

    previously sorted values

    >>> ml = MyList()
    >>> ml.append(1)
    >>> ml.append(2)
    >>> ml.append(3)
    >>> ml.print_sorted()
    [1, 2, 3]
