The my-list module
==================

using print_sorted
------------------

This is python doctest module to test a function that prints a
sorted list

    >>> MyList = __import__('1-my_list').MyList
    >>> ml = MyList()
    >>> ml.append(3)
    >>> ml.append(3)
    >>> ml.append(3)
    >>> ml.print_sorted()
    [3, 3, 3]
    >>> ml.append(4)
    >>> ml.append(1)
    >>> ml.print_sorted()
    [1, 3, 3, 3, 4]
